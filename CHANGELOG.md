# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Monorepo foundation with Turborepo and pnpm workspaces
- NestJS 11 backend with modular architecture
- Prisma ORM with 30+ database models
- Authentication module (JWT, RBAC, account locking, session management)
- Employee management (CRUD, search, org chart)
- Attendance tracking (clock-in/out, overtime, monthly summaries)
- Leave management (apply, approve/reject, balance tracking)
- Policy engine with version control and rule evaluation
- Health diagnostics (DB, Redis, memory monitoring)
- Audit logging (immutable, append-only)
- Notifications service (multi-channel, read tracking)
- Next.js 15 frontend with App Router
- CSS design system (light/dark themes, 15+ component classes)
- Login page with split-screen premium design
- Dashboard shell (collapsible sidebar, sticky topbar)
- Dashboard home (stat cards, activity table, quick actions)
- Shared package for types and constants
- Docker Compose (PostgreSQL, Redis, Elasticsearch, MinIO, MailHog)
- Database seed (roles, permissions, departments, leave types, admin user)
- Git hooks (Husky + commitlint + lint-staged)
- Conventional Commits enforcement
- EditorConfig for cross-IDE consistency
- Contributing guidelines with Git Flow branching strategy
